Welcome to my website

I'd like to think of myself as a programmer of some sort, so I've created this site to have information about all significant projects I've been involved in consolidated in one place and readily available for public display.
For now it's going to be just short descriptions and links, but I might also start displaying small, interactive demos here once I get better at web programming.

I understand not everyone is comfortable with allowing JavaScript to run on just any random site they visit, so I've put extra effort to make sure the site works smoothly without any JavaScript at all.

This might change in the future if I start showcasing more complex demos here or I simply decide it takes too long to maintain so many separate pages.
